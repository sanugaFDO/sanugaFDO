## 🚀 Professional Summary

I am a **Full-Stack Software Engineering student at Birmingham City University**, passionate about bridging the gap between digital interfaces and physical hardware. My engineering focus spans from architecting intuitive desktop and web management systems to developing intelligent, sensor-driven IoT solutions. 

Whether I am designing complex user interfaces in Python, configuring electro-pneumatic automation circuits, or programming ESP32 microcontrollers for autonomous robotics, I thrive on turning complex technical challenges into streamlined, real-world solutions.

**🔭 Current Focus Areas:**
* **Full-Stack Development:** Building end-to-end applications, database integrations, and comprehensive UI systems (React, Python, PHP).
* **IoT & Embedded Systems:** Designing smart hardware solutions and WiFi-controlled robotics using Arduino and ESP32.
* **Industrial Automation:** Exploring pneumatic logic, PLC programming, and sequential control systems.
